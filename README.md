# DCS Converter

Excel 试验条件表 → Markdown 转换工具

## 功能简介

将 DCS（分散控制系统）的试验条件 Excel 表格自动转换为结构清晰的 Markdown 文档，支持多层级逻辑条件的智能格式化。

### 核心特性

- **拖拽/点击导入** — 支持 `.xlsx` / `.xls` 格式
- **多 Sheet 预览** — 左侧目录树一键切换
- **智能层级识别** — 自动识别条件嵌套层级并格式化
- **合并输出** — 可将所有 Sheet 合并为一个 Markdown 文件
- **一键复制/下载** — 快速导出结果

### 条件层级格式化规则

| 层级 | 格式 | 示例 |
|------|------|------|
| Level 1（单条件） | 每项单独一行 | `- 条件1` |
| Level 2（两层逻辑） | OR/AND 组各占一行 | `- 条件1或条件2` |
| Level 3（三层逻辑） | 括号包裹嵌套组 | `- (条件1或条件2）与（条件3或条件4）` |

## 快速开始

### 方式一：本地服务器（推荐）

```bash
# 需要 PowerShell 5.1+
.\server.ps1
```

浏览器自动打开 `http://localhost:18888/`

### 方式二：直接打开

双击 `dcs_converter.html` 在浏览器中打开（需联网加载 xlsx 解析库）

## Excel 表格格式要求

程序按以下列结构解析数据：

| A列 | B列 | C列 | D列 | E列 |
|-----|-----|-----|-----|-----|
| 序号 | 分隔符/条件 | 分隔符/条件 | 条件 | 条件 |

- **分隔符**：`与`（AND）或 `或`（OR）
- **A列**：条目序号（如 `1`、`2`）
- **子标题**：以 `一、` `二、` 等开头的行作为 `##` 标题
- **分组标题**：以 `数字.` 开头的行作为 `###` 标题

### 示例

| A | B | C | D |
|---|---|---|---|
| 1.自动开条件(与) | | | |
| 序号 | 条件内容 | | |
| 1 | 输煤急停（取反） | | |
| 2 | 或 | 输煤联锁模式 | |
| | 与 | 输煤自动模式 | |

转换结果：

```markdown
### 1.自动开条件(与)

- 输煤急停（取反）
- 输煤联锁模式或输煤自动模式
```

## 使用步骤

1. 启动服务或打开 HTML 文件
2. 点击上传区域或拖拽 Excel 文件到页面
3. 左侧目录树选择要预览的 Sheet
4. 点击「全部转换」生成 Markdown
5. 点击「复制」或「下载 .md」导出结果

## 技术栈

- 纯前端实现，无需后端
